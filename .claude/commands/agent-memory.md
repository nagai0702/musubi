# メモリ保存スキル（Agent Memory）

会話中の重要な情報をメモリファイルに保存する。

## 引数
$ARGUMENTS - ユーザーが「覚えておいて」と言った内容

## 実行手順

1. **保存すべき情報を抽出**
   - $ARGUMENTS から保存対象の情報を特定
   - 以下のカテゴリに分類:
     - **decisions** - 意思決定とその理由
     - **preferences** - ユーザーの個人的な好み・方針
     - **facts** - 事実情報・データ・数値
     - **contacts** - 人物情報・連絡先
     - **procedures** - 手順・やり方

2. **カテゴリに応じたファイルに追記**
   - 保存先: `00_context/memories/` 配下
   - ファイル名:
     - `decisions.md` - 意思決定ログ
     - `preferences.md` - 好み・方針
     - `facts.md` - 事実情報
     - `contacts.md` - 人物情報
     - `procedures.md` - 手順メモ
   - ファイルが存在しない場合は新規作成

3. **保存フォーマット**
   - 日付とともに追記
   - 既存の内容と重複する場合は更新（上書きではなく追記）

4. **保存内容をユーザーに報告**
   - 何を、どのファイルに保存したかを表示

## 保存フォーマット

```markdown
### YYYY-MM-DD
- [保存した内容]
  - 背景/理由: [あれば記載]
```

## 出力フォーマット

```
メモリに保存しました:
- ファイル: 00_context/memories/[ファイル名]
- カテゴリ: [カテゴリ名]
- 内容: [保存した内容の要約]
```

## 注意事項
- 機密性の高い情報（パスワード、APIキー等）は保存しない
- 既存のメモリと矛盾する情報がある場合は、ユーザーに確認する
- メモリファイルは定期的に整理が必要であることをユーザーに伝える
