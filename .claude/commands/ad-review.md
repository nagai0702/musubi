# 広告運用レビュースキル

広告運用チームを起動し、META広告のパフォーマンス分析と改善提案を行う。

## 引数
$ARGUMENTS - 分析の焦点やリクエスト（省略時は全体レビュー）

## 実行手順

`.claude/agents/ad-operations-team.md` に定義されたチーム構成に従い、以下の順序で実行する。

### Phase 1: データ収集・分析（並列）
以下の3つの分析を並列で実行する：

**A. マーケット分析**
- Web検索で結婚相談所業界の最新トレンドを調査
- 競合の広告訴求ポイントを分析

**B. ターゲティング分析**
- META広告MCPを使い、現在のオーディエンス設定を取得
- ターゲティングの改善案を策定

**C. パフォーマンス分析**
- META広告MCPを使い、直近のキャンペーンデータを取得
- CPA, ROAS, CTR, CVR等のKPIを分析
- 広告セット別・クリエイティブ別の成績を比較

### Phase 2: 戦略統合
- Phase 1の3つの分析結果を統合
- 広告戦略の方向性を決定
- 予算配分の最適化案を策定

### Phase 3: クリエイティブ提案
- 戦略に基づいた新しい広告コピーを3パターン作成
- 各パターンの訴求軸を明確化
- A/Bテストの設計

### Phase 4: 運用プラン策定
- 以下の4軸でレビュー：
  1. ターゲットと訴求の整合性
  2. 予算配分の妥当性
  3. 期待CPA・ROAS
  4. リスクと対策
- 最終的な運用プランを策定

## 出力
- 分析結果を `output/trends/ad-analysis-YYYY-MM-DD.md` に保存
- 運用プランを `output/articles/ad-plan-YYYY-MM-DD.md` に保存
- チャットにサマリーを表示
